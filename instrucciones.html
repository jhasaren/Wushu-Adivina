Instrucciones Wushu!!
  • Elige una categoría
  • Acepta, si eres Mayor de +18 años
  • Agarra el dispositivo con las dos manos de manera Horizontal
  • Colocalo a nivel de tú frente, para que los demás puedan ver la pantalla
  • Espera unos segundos mientras cargan las palabras
  • ¡Juega! Se mostrará una palabra para Adivinar. Puedes Actuar o Dibujar
  • Para cambiar de palabra se debe girar el dispositivo hacia Arriba
  • Cada sesión durará 60 segundos, ó para la categoría Dibujo 120 segundos
  • Gana quien adivine la mayor cantidad de palabras posible por sesión
