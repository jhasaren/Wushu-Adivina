# Wushu-Adivina
Elige entre categorías y palabras para que puedan ser adivinadas antes que se acabe el tiempo. Puedes utilizar técnicas de actuación y/o dibujo en pizarra para adivinar.

Solo debes elegir una categoría y esperar que automáticamente se carguen las palabras en pantalla, se mostrara una palabra a la vez la cual podrás cambiar girando el dispositivo hacia arriba. No requiere conexión a Internet para Jugar, pues las palabras son precargadas en el dispositivo al momento de la instalación.

Características:

*Juega en familia o con amigos, en grupos o parejas.
*Fácil uso, gira hacia arriba el dispositivo para cambiar de palabra.
*Efectos de sonido incorporados para mejor experiencia.
*No requiere conexión a Internet para Jugar.

Instrucciones:

* Arma grupos de personas, mínimo dos personas por cada grupo.
* Elige una categoría
* Acepta, si eres mayor de +18 años
* Agarra el dispositivo con las dos manos de manera Horizontal
* Colócalo a nivel de tú frente, de manera que los demás puedan ver la pantalla
* Espera unos segundos mientras cargan las palabras
* Juega!! Se mostrara una palabra para Adivinar
* Para cambiar de Palabra se debe girar el dispositivo hacia arriba
* Cada sesión durara 60 segundos, las de Dibujo 120 segundos.
* Gana el equipo que adivine la mayor cantidad de palabras posible por sesión

Categorías Disponibles:

*Actualo
*Deportes
*Animales
*Dibujalo

Se recomienda Solo para mayores de +18 años, pues en algunas categorías el lenguaje puede ser fuerte.
